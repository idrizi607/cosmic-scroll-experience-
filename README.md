An immersive, scroll-driven digital experience featuring celestial aesthetics and advanced motion logic. This project demonstrates high-performance frontend animations and storytelling through code.

---

## ✨ The Experience
This project pushes the boundaries of traditional web layouts by treating the browser viewport as a cinematic stage. As the user scrolls, elements transition, rotate, and scale based on precise scroll-trigger positions.

### Key Technical Features:
* **GSAP ScrollTrigger:** Orchestrated complex timelines synchronized with the user's scroll speed.
* **Celestial Parallax:** Layered depth effects using high-resolution cosmic imagery.
* **Dynamic Content Reveal:** Sophisticated "fade-and-slide" entrance animations for UI elements.
* **Performance Optimized:** Smooth 60fps transitions and hardware-accelerated transforms.

---

## 🛠️ The Tech Stack
* **Engine:** GSAP (GreenSock Animation Platform)
* **Logic:** Vanilla JavaScript (ES6+)
* **Styling:** CSS3 (Custom Properties & Flexbox)
* **Structure:** Semantic HTML5

---

## 🚀 Key Learning Milestones
1. **Mathematics of Motion:** Calculating scroll-start and scroll-end triggers for pixel-perfect timing.
2. **Resource Management:** Handling large background assets while maintaining fast load times.
3. **Responsive Animation:** Ensuring the cosmic flow remains consistent across mobile and desktop screens.
